# 정보는 뭔가요?
## Information
- Claude Elwood Shanon : 클로드 섀넌
- a quantitative representation that measures the uncertainty.
- = 불확실한 상황을 측정해서 수치적으로 표현한 것
$$I(x)=-log_{2}P(x)$$
$x$ : 사건 $I$ : 정보량 $P$ : 확률<br>
$I$의 단위 : $bit$ = binary digit
